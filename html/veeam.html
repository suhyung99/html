Veeam백업

 

단일 통합 데이터 관리 플랫폼

글로벌 시장 점유율 1위

2022년 하반기 Veeam은 백업 시장 점유율 1위 달성 기업

 

전세게 시장 점유율 No.1 쿠버네티스 백업

 

1. 하이브리드 클라우드 데이터 보호 및 재해복구

2. 물리서버 OS 이미지 백업 및 마이그레이션

3. 초고속 NAS백업

4. 변경 불가한 데이터 장기보관

5.쿠버네티스 백업

6. Microsoft 365 백업

 

Why? Veeam 

1. 하이브리드 클라우드 데이터 보호 및 재해복구


- 다양한 하이퍼바이저 Agengless백업

- 즉시복구(Instance Recovery)

- 손쉬운 DB복구

- 이기종 플랫폼 마이그레이션


2. 물리 서버 OS이미지 백업 및 마이그레이션


- OS + DB백업

- Instance Recovery

- P2P, P2V, P2C, V2P, C2P


3. 초고속 NAS백업


- 스캔 타임을 없앤 초고속 백업(2백2십만 파일 17분 전체 백업)

- Forever 증분백업

4. 변경 불가한(Immutable)데이터 장기 보관


- 랜섬웨어로부터 완벽 데이터 보호

- Retention Lock

 

5. 쿠버네티스 백업 (전세게 시장점유율 1위)


- 컨테이너 환경의 백업과 재해복구 관리

- Veeam Kasten, 최고의 쿠버네티스 전용백업 솔루션

- 최다 레퍼런스 보유(레드햇 오픈시프트, 매텍 아코디언, VMware탄주, 디아만티, 나무기술 칵테일, 티맥스 하이퍼클라우드,수세랜처, 뉴타닉스 카본, HPE 에즈메탈,,,)

- Agentless, Pods탑재 불필요


6. Microsoft 365백업


- M365 최적 백업 솔루션

- M365 자체 기능만으로 계획된 악의적 데이터 삭제와 파손에
충분할수 없음.

- 최적 M365 백업 솔루션은 Veeam Backup for M365
